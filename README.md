# init

## FrontEnd

```bash
# vite를 사용하여 frontend 설정하기
$ npm create @vite/latest
# client로 이동하여 nodemodule 설치
$ cd client
```

## BackEnd

```bash
# package.json 설정
$ npm init -y
$ npm i express
$ npm i -D nodemon
$ npm i morgan
$ npm i --save @types/morgan
# TypeScript에서 express 인식
$ tsc --init
# TypeScript가 Node.js에서 실행하도록 설정
$ npm i -D typescript ts-node tsx @types/node @types/express
$ npm i mongoose
$ npm i dotenv
$ npm i -D @types/dotenv
$ npm i @types/bcrypt
$ npm i jsonwebtoken
$ npm i --save @types/jsonwebtoken
$ npm i cookie-parser
$ npm i --save @types/cookie-parser
```
